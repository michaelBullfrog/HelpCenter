# TS Calling

Use this playbook for Webex Calling issues.

## Common symptoms

- Poor audio quality
- Dropped calls
- One-way audio
- Calls not routing correctly
- Outbound caller ID issues
- Voicemail not working
- Device not ringing
- Number assignment or porting issues

## First checks

1. Confirm the affected user, extension, number, location, and device.
2. Confirm whether the issue affects one user, one site, or multiple locations.
3. Check Control Hub user calling settings.
4. Check device registration status.
5. Validate PSTN/location configuration.
6. Capture timestamps, calling/called numbers, and call examples.

## Escalation packet

Include:

- User email
- Calling number and called number
- Direction: inbound, outbound, internal, transferred
- Time and timezone
- Device type
- Network path/VPN status
- Screenshots of relevant Control Hub settings
