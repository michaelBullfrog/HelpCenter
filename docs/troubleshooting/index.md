# Troubleshooting Overview

Use this section as the main troubleshooting hub.

## Troubleshooting tabs

The troubleshooting content is split into dedicated top-level tabs:

- **TS Calling** - Webex Calling call quality, routing, numbers, voicemail, and user feature issues.
- **TS Contact Center** - Entry points, queues, flows, Agent Desktop, supervisor tools, and reporting issues.
- **TS Devices** - Desk phones, RoomOS devices, workspaces, registration, and device health.
- **TS APIs** - OAuth, tokens, scopes, rate limits, payloads, and webhook issues.
- **TS Meetings** - Meeting join, audio, video, recording, and transcription issues.

## Standard troubleshooting flow

1. Confirm the user, location, device, service, and time of issue.
2. Identify the affected product area.
3. Check whether the issue is isolated or widespread.
4. Gather logs, examples, screenshots, and timestamps.
5. Validate configuration in Control Hub.
6. Reproduce or test if possible.
7. Escalate with a complete packet when needed.
