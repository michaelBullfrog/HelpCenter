# TS Devices

Use this playbook for Webex devices, phones, and workspaces.

## Common symptoms

- Phone not registering
- Activation code not working
- Device offline in Control Hub
- Room device not pairing
- Firmware or upgrade issue
- Incorrect workspace assignment
- Audio/video peripheral issue

## First checks

1. Confirm device model and serial/MAC address.
2. Confirm whether it is assigned to a user or workspace.
3. Check network connectivity, DNS, NTP, and firewall path.
4. Verify activation state in Control Hub.
5. Reboot only after collecting the current status and error message.

## Escalation packet

Include:

- Device model
- Serial number or MAC address
- Workspace/user assignment
- Location
- Firmware version
- Network details
- Photos or screenshots of device errors
