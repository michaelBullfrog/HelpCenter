# TS Meetings

Use this playbook for Webex Meetings issues.

## Common symptoms

- Cannot join a meeting
- Audio or video failure
- Recording not available
- Transcription issue
- Scheduling issue
- Host/cohost permission issue
- Meeting link or calendar invite problem

## First checks

1. Confirm meeting number/link and site.
2. Confirm user role: host, cohost, attendee, scheduler.
3. Check whether the issue is browser, desktop app, mobile app, or device-based.
4. Validate network access and browser permissions.
5. Gather exact time, timezone, and screenshots.

## Escalation packet

Include:

- Meeting number or link
- Webex site
- Host email
- Affected user email
- Platform/app version
- Time of issue
- Error message or screenshot
