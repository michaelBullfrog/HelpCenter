# Source Quality Rules

To keep this knowledge base accurate, use this source priority:

1. Official Cisco/Webex documentation
2. Webex Developer documentation
3. Cisco release notes and changelogs
4. Cisco Community accepted answers
5. Internal validated lab notes
6. Customer-specific notes

!!! warning "Do not override official docs"
    Internal notes are useful, but they should not override current Cisco/Webex documentation unless the article clearly explains why.
