## First questions

- Is the issue inbound, outbound, internal, external, voicemail, device registration, media, or call routing?
- Is it affecting one user, one location, one device model, or the whole organization?
- When did it start?
- What changed recently?

## Evidence to collect

- Calling user or workspace
- Calling number and called number
- Date/time with timezone
- Call examples
- Device model and firmware
- Network path and public IP
- Error messages or screenshots
