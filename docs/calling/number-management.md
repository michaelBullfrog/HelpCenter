## Purpose

Use this page to document number assignment, location-level number management, extensions, DID assignment, toll-free numbers, and porting notes.

## Checklist

- Confirm customer organization.
- Confirm location.
- Confirm number type: DID, toll-free, extension-only, main number, or alternate number.
- Confirm whether the number is assigned, available, or pending port.
- Validate inbound and outbound call behavior.

## Troubleshooting notes

- If a number is not searchable, normalize the format: E.164, national format, and digits-only.
- Confirm the number is in the correct location.
- Confirm the number has not been assigned to a user, workspace, call queue, auto attendant, hunt group, or virtual line.
