## What Webex Calling is

Webex Calling is Cisco's cloud calling solution. It replaces or supplements legacy PBX functionality with cloud-managed calling features administered through Control Hub.

## Core building blocks

| Building block | Purpose |
| --- | --- |
| Locations | Define site-level calling context. |
| Numbers | Assign PSTN numbers to users, services, or workspaces. |
| Users | People with calling licenses and calling features. |
| Workspaces | Shared rooms or spaces with calling-enabled devices. |
| Devices | Phones, room devices, and Webex App endpoints. |
| Auto attendants | Menu-based routing for main numbers or departments. |
| Call queues | Queue and distribute calls to agents/users. |
| Hunt groups | Ring groups of users according to a pattern. |
| Schedules | Define business hours, holidays, and routing behavior. |

## Support-first troubleshooting order

1. Identify the affected caller/called party.
2. Confirm whether the issue affects one user, one location, one number, or everyone.
3. Check licensing, number assignment, and location.
4. Check device or Webex App registration.
5. Check routing features such as auto attendants, call queues, hunt groups, forwarding, DND, and schedules.
6. Check call quality/troubleshooting tools in Control Hub.
7. Escalate with call examples and timestamps.
