# Users and Licensing

**Product area:** Control Hub  
**Audience:** Webex administrators, support teams  
**Last reviewed:** 2026-06-04  
**Primary source:** Webex Help Center — Get started with Control Hub: <https://help.webex.com/en-us/article/nkhozs6/get-started-with-control-hub>

## Overview

Users are managed in Control Hub and assigned licenses for the Webex services they need. Licensing affects whether a user can use Webex Calling, Meetings, Messaging, Contact Center, or other entitled features.

## Standard user review checklist

When troubleshooting a user issue, check:

- User exists in the correct organization.
- User email address is correct.
- User has accepted or completed activation if required.
- Required license is assigned.
- User is assigned to the correct location, when applicable.
- Calling extension or phone number is assigned if the issue is Calling-related.
- Contact Center profile/team settings are correct if the issue is Contact Center-related.

## Common user/license issues

| Symptom | Possible cause |
| --- | --- |
| User cannot access a feature | License missing or incorrect. |
| User cannot make/receive calls | Calling license, number, extension, location, or device assignment issue. |
| User cannot access Contact Center | Contact Center license or user profile/team issue. |
| Admin cannot see service settings | Admin role issue, not necessarily licensing. |

## Documentation note

When writing user/license articles, include both the admin action and the validation step. A correct license assignment is not complete until the user can successfully use the service.
