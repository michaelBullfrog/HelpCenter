## Purpose

Use this page to document which administrative roles can manage users, services, Contact Center features, and organization-level settings.

## Key guidance

- Confirm whether the admin is internal, partner, external, full admin, read-only admin, provisioning admin, or Contact Center service admin.
- Validate the user has the correct product license before assuming a role grants feature access.
- For Contact Center, distinguish between Control Hub access and Management Portal or Agent Desktop access.

## Validation checklist

- User appears under Control Hub users.
- Correct license is assigned.
- Correct admin role is assigned.
- User can access the expected product area.
- User cannot access restricted areas beyond their role.
