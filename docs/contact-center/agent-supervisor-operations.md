## Purpose

Document day-to-day agent and supervisor workflows, including sign-in, states, queue assignment, monitoring, reporting access, and basic troubleshooting.

## Common support checks

- Agent has the correct license.
- Agent is assigned to the correct team.
- Agent profile includes the expected queues and desktop profile.
- Supervisor has the required role and team visibility.
- Business hours and routing are active.
