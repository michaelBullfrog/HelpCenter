# Glossary

| Term | Definition |
|---|---|
| Control Hub | Webex administrative portal for managing users, services, devices, policies, and organization settings. |
| Webex Calling | Cisco cloud calling service managed through Control Hub. |
| Webex Contact Center | Cisco cloud contact center platform for queues, agents, supervisors, flows, reporting, and customer interactions. |
| Entry Point | Contact Center object that receives inbound interactions before routing logic sends them onward. |
| Queue | Contact Center object used to hold and route interactions to available agents. |
| Flow | Contact Center call or interaction logic used to route and process customer interactions. |
| OAuth Integration | Webex app pattern that authorizes API calls on behalf of a user. |
| Webhook | Event subscription that sends notifications to a target URL when matching Webex events occur. |
