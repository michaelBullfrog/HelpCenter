# Phase 1B Changelog - Troubleshooting Tabs

This update makes troubleshooting easier to use by adding dedicated top navigation tabs for each troubleshooting category.

## Added

- TS: Calling
- TS: Contact Center
- TS: Devices
- TS: APIs
- TS: Meetings

## Updated

- Troubleshooting home page
- Escalation checklist
- Ticket template
- MkDocs navigation

## Why this matters

MkDocs Material uses top-level navigation items as tabs when `navigation.tabs` is enabled. By promoting each troubleshooting category to a top-level navigation item, each troubleshooting area now appears as its own tab across the top of the site.
